# R-5000 タグリーダー 出力設定メモ

送信出力（dBm）と読み取り範囲の目安。

| 出力（dBm） | 読み取り範囲（目安） |
|------------|---------------------|
| 0 dBm      | 約 3 cm             |
| 5 dBm      | 約 5 cm             |
| 10 dBm     | 約 15〜18 cm        |
| 15 dBm     | 約 30 cm            |
| 20 dBm     | 約 50 cm            |
| 30 dBm     | 約 3 cm             |

※ 30 dBm の「3 cm」は要確認（0 dBm と同じ値のため、3 m などの誤記の可能性あり）。
