# タグリーダー実機テスト手順（Android）

目的: デモ機到着後に **最短で「接続→読取→一覧に表示」** まで確認する。

## 事前準備（PC）

- `tools/setup_tss_sdk.ps1` を実行し、以下が配置されていること
  - `android/app/libs/TSS_SDK.aar`
  - `android/app/src/main/jniLibs/<arch>/*.so`（例: `arm64-v8a/libDeviceAPI.so`）

## 事前準備（Android端末）

- 端末の「設定」→「Bluetooth」から、タグリーダーを **ペアリング**しておく
- 初回起動時に表示される **Bluetooth権限** は許可する
  - Android 12+ は `Bluetooth 接続/スキャン`
  - Android 6〜11 は `位置情報` を求められる場合がある（スキャン要件）

## アプリでの最短確認

### 1) 接続確認

1. メイン → **「タグリーダー接続」**
2. **「スキャン」** を押す（ペアリング済み一覧が表示される）
3. 対象デバイスの **「接続」** を押す
4. 画面上部が **「接続中: <デバイス名>」** になればOK

### 2) タグ読取（一覧）

1. メイン → **「ICタグ読取・更新」**
2. **「読取開始」** を押す
3. タグを近づけ、一覧に **EPC が追加**されることを確認

### 3) タグ読取（詳細）

1. メイン → **「タグ情報表示」**
2. **「読取開始」** を押す
3. `RSSI/CH/TIME` 等の付加情報が表示されることを確認

### 4) 電波強度の変更

1. メイン → **「タグリーダー接続」** でタグリーダーが接続済みであることを確認（「接続中: ...」と表示）
2. メイン → **「設定」** → **「出力設定」**
3. 「電波強度」画面で現在の **dBm / mW** が表示されることを確認
4. `+10 / +1 / -1 / -10` ボタンを押すと、メーターおよび dBm 表示が変化する
5. 同じデバイスで TSS 提供のデモアプリなどを開き、電波強度設定が同じ値に変わっていること、または読取距離が変化することを確認
6. アプリを再起動し、再度 **「電波強度」** 画面を開いて、前回設定した dBm が維持されていることを確認

## うまくいかない時の切り分け

- **一覧にデバイスが出ない**
  - 端末側でペアリング済みか確認
  - 権限が拒否になっていないか確認（設定→アプリ→権限）
- **接続できない**
  - タグリーダーが他端末に接続中でないか
  - BluetoothがONか
- **接続はできるが読めない**
  - 読取距離/タグの向き
  - 端末の省電力設定（Bluetooth制限）
  - タグリーダー側のトリガ/出力設定（必要ならSDKデモアプリで切り分け）
